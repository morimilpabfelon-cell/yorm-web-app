# Seguridad de Yorm Pay

## Estado actual

Foundation 0 es exclusivamente sandbox. No debe procesar, custodiar ni simular como reales fondos de usuarios.

## Prohibido en esta fase

- credenciales reales de proveedores;
- secretos dentro del repositorio;
- datos personales reales;
- transferencias bancarias reales;
- emisión o procesamiento real de tarjetas;
- cripto o USDT real;
- webhooks públicos sin autenticación;
- despliegues de producción.

## Reportes

Los hallazgos de seguridad deben documentarse en un issue privado o canal autorizado antes de publicar detalles explotables.
